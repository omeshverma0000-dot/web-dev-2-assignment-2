# 🌦️ Weather Tracker Web App

A simple and interactive Weather Tracker application built using HTML, CSS, and JavaScript. This app fetches real-time weather data using an API and provides a clean user interface with additional features.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Displays temperature and weather condition
- 🌤️ Dynamic weather icons
- 🎨 Light/Dark mode toggle
- 🕘 Search history using LocalStorage
- ❌ Option to delete history items
- 🎯 Input validation and error handling
- 🎨 Clean and responsive UI

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API
